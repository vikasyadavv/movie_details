# 🎬 Enhanced iPhone Movie Shortcut - Complete Data Guide

## 📊 All Available Movie Information from OMDb API

### 🎯 Basic Info (Currently Used)
- ✅ **Title** - Movie name
- ✅ **Year** - Release year
- ✅ **imdbRating** - IMDB score (1-10)
- ✅ **imdbVotes** - Number of IMDB votes
- ✅ **Genre** - Movie genres
- ✅ **Runtime** - Duration in minutes
- ✅ **Director** - Director name
- ✅ **Plot** - Movie description

### 🆕 Additional Information You Can Add

#### 📽️ **Production Details**
- **Writer** - Screenplay writers
- **Actors** - Main cast (first 3-4 actors)
- **Production** - Production company
- **Country** - Country of origin
- **Language** - Primary language

#### 🏆 **Ratings & Awards**
- **Rated** - MPAA rating (PG, PG-13, R, etc.)
- **Metascore** - Metacritic score (0-100)
- **Awards** - Oscar wins, nominations, etc.
- **Ratings** - Multiple rating sources (IMDB, Rotten Tomatoes, Metacritic)

#### 💰 **Box Office & Release**
- **Released** - Release date
- **BoxOffice** - Box office earnings
- **DVD** - DVD release date
- **Website** - Official website

#### 🖼️ **Visual Content**
- **Poster** - Movie poster URL
- **Type** - Content type (movie, series, episode)

#### 🔗 **Technical Details**
- **imdbID** - IMDB identifier
- **Response** - API success status

### 📱 **Version 3: Poster Display**
For iOS 14+, you can display the movie poster:

1. **Add "Get Value for" action**:
   - Dictionary Key: `Poster`
   - Variable Name: `PosterURL`

2. **Add "Get Contents of URL" action**:
   - URL: Use `PosterURL` variable
   - Variable Name: `PosterImage`

3. **Add "Quick Look" action**:
   - Input: `PosterImage`

### 📱 **Version 4: Multiple Ratings**
The `Ratings` field contains an array with multiple rating sources:

1. **Add "Get Value for" action**:
   - Dictionary Key: `Ratings`
   - Variable Name: `AllRatings`

2. **Add "Get Item from List" actions** for each rating:
   - Item 1: IMDB rating
   - Item 2: Rotten Tomatoes rating  
   - Item 3: Metacritic rating

## 🛠️ **Advanced Features You Can Add**

### 🔍 **Search by IMDB ID**
Change API URL to: `http://www.omdbapi.com/?apikey=KEY&i=IMDBID`

### 📺 **TV Series Support**
The API also supports TV series - just search for show names!

### 🎯 **Year-Specific Search**
Add year parameter: `http://www.omdbapi.com/?apikey=KEY&t=TITLE&y=YEAR`

### 📄 **Full Plot**
For longer plot: `http://www.omdbapi.com/?apikey=KEY&t=TITLE&plot=full`
